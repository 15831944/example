# Copay

下载源码

```shell
git clone https://github.com/bitpay/copay.git
cd copay
```

运行

```shell
npm run apply:copay
npm run start
```

Web

`localhost:8100`

Anroid

```shell
npm run apply:copay
npm run prepare:copay
npm run start:android
```

iOS

```shell
npm run apply:copay
npm run prepare:copay
npm run start:ios
```
